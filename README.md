# port2nix

## Introduction

`port2nix` is a toolbox for nix users who want to port configs in other languages to nix.

## Supported Languages

- [ ] toml
- [ ] yaml
- [ ] json

## Development

1. Ensure `nix` is installed and `flakes`, `nix-command` are enabled.
2. `nix develop`
3. `npm install`
4. `npm run dev` to run dev server.
